# Neofetch config 

## Setup

1. Install Neofetch
```
https://github.com/dylanaraps/neofetch/wiki/Installation
```

2. Clone this repo into `~/.config/neofetch`

``` 
git clone https://github.com/Grubblex/neofetch_config.git
```

3. Run `neofetch` inside Terminal
